# Instalasi

## Windows

1. Download dbeaver [di sini](https://dbeaver.io/)

2. Download sesuai sistem operasi yang digunakan
   ![](./assets/dbeaver-download-win.gif)

3. Cara instalasinya cukup mudah. Tinggal next saja dan disesuaikan dengan kebutuhan. Atau bisa ikuti petunjuk dibawah
   ![](./assets/dbeaver-install-win.gif)


## Ubuntu

1. Kamu bisa download melalui ubuntu software
2. Cari `dbeaver` kemudian klik install
![](./assets/dbeaver-install-ubuntu.png)

## Konek ke mysql

1. Buka dbeaver yang sudah di install. Saat pertama kali dibuka maka tampilannya akan seperti di bawah ini.
![](./assets/dbeaver-connect.png)

2. Pilih mysql sebagai database yang akan dihubungkan.
3. Isi segala inputan yang dibutuhkan seperti lokasi server dan user mysql. Jika sudah, klik test connection untuk memeriksa apakah sudah berhasil terkoneksi atau masih membutuhkan driver tambahan
![](./assets/dbeaver-connect2.png)

4. Jika sudah berhasil terkoneksi, maka tampilannya akan seperti ini
![](./assets/dbeaver-connect3.png)
